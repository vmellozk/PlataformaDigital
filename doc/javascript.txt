document: refere-se ao objeto do DOM (Document Object Model) que representa a página web carregada.
É a interface principal para interagir com o conteúdo da página.

addEventListener: um método que adiciona um ouvinte de eventos ao objeto especificado. Neste caso,
está sendo adicionado um ouvinte para o evento DOMContentLoaded, que dispara quando o documento HTML
foi completamente carregado e analisado, sem esperar por folhas de estilo, imagens e subframes.

'DOMContentLoaded': um tipo de evento que indica que o documento HTML foi totalmente carregado e pode
ser manipulado.


const: uma palavra-chave que declara uma constante, ou seja, uma variável cujo valor não pode ser
reatribuído. É usada para proteger a variável de modificações acidentais.

fields: o nome da variável que contém um array. Um array é uma estrutura de dados que pode armazenar
múltiplos valores em uma única variável.


forEach: um método de array que executa uma função para cada elemento do array. O parâmetro field
representa o elemento atual sendo processado.

field => {: uma função de flecha que será chamada para cada elemento do array. Dentro do escopo dessa
função, field representa o objeto atual do array.


document.getElementById(...): um método que retorna o elemento do DOM que possui o id correspondente
ao valor passado como argumento. Aqui, field.id é usado para pegar o id do objeto atual do array fields.


const counterElement = document.getElementById(field.counterId); Similar à linha anterior, mas aqui estamos
buscando o elemento que será usado para exibir o contador de caracteres restantes, usando o counterId do
objeto field.

inputElement.addEventListener(...): adiciona um ouvinte de eventos ao elemento de entrada (inputElement).
O evento input é disparado sempre que o valor do campo é alterado (por exemplo, ao digitar).

const remaining: declara uma constante chamada remaining.

field.maxLength: acessa a propriedade maxLength do objeto field, que define o número máximo de caracteres
permitidos.

inputElement.value.length: obtém o comprimento (número de caracteres) do valor atual do campo de entrada.
A subtração calcula quantos caracteres ainda podem ser inseridos.

counterElement.textContent: altera o conteúdo de texto do elemento do contador. textContent é uma
propriedade que define ou obtém o texto contido no elemento.

`${remaining} caracteres restantes`: uma template string (string template) que permite interpolar
expressões dentro de uma string. Aqui, a variável remaining é incluída na string que será exibida.
